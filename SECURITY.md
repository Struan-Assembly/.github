# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in any Struan Assembly project, please report it responsibly. Do not open a public issue.

Email **security@struanassembly.com** with:

- A description of the vulnerability
- Steps to reproduce
- Affected project and version (if known)

We will acknowledge your report within 3 business days and provide updates as we investigate. We ask that you give us reasonable time to address the issue before any public disclosure.

## Supported Versions

Only the latest release of each project receives security updates. If you are running an older version, please upgrade before reporting.
